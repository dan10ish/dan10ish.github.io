<h2>Matrices</h2>
<ul>
<li>Matrix multiplication <span class="math math-inline">AB</span> between two <span class="math math-inline">n \times n</span> matrices results in another <span class="math math-inline">n \times n</span> matrix, where each element is calculated as:
<div class="math math-display">(AB){ij} = \sum{k=0}^{n-1} A_{ik} \times B_{kj}</div>
</li>
</ul>
<h4>1. Frievald's Algorithm</h4>
<p>To check matrix multiplication, <span class="math math-inline">AB = C</span> in <span class="math math-inline">O(n^2)</span> instead of <span class="math math-inline">O(n^3)</span> by randomly choosing a vector <span class="math math-inline">x</span> and testing <span class="math math-inline">A(Bx) = Cx</span>.</p>
<pre><code class="language-python">from random import randint
from sys import stdin

def readint():
  return int(stdin.readline())

def readarray(typ):
  return list(map(typ, stdin.readline().split()))

def readmatrix(n):
  M = []
  for _ in range(n):
    row = readarray(int)
    assert len(row) == n
    M.append(row)
  return M

def mult(M, v):
  n = len(M)
  return [sum(M[i][j] * v[j] for j in range(n)) for i in range(n)]

def freivalds(A, B, C):
  n = len(A)
  x = [randint(0, 1000000) for j in range(n)]
  return mult(A, mult(B, x)) == mult(C, x)

if __name__ == "__main__":
  n = readint()
  A = readmatrix(n)
  B = readmatrix(n)
  C = readmatrix(n)
  print(freivalds(A, B, C))
</code></pre>
<p>Input:</p>
<pre><code class="language-txt">2      # value of n in n by n matrices
1 2    # a11 a12
3 4    # a21 a22
5 6    # b11 b12
7 8    # b21 b22
9 10   # c11 c12
11 12  # c21 c22
</code></pre>
<p>Output: <code>False</code></p>
<p>TC : <span class="math math-inline">O(n^2)</span> &#x26; SC : <span class="math math-inline">O(n)</span></p>
