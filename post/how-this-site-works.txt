b:T1aad,<h1>Introduction</h1>
<p>I rebuilt my website with <code>Next.js 14</code>, focusing on minimalism and performance while keeping powerful features like math rendering, syntax highlighting, and dynamic theming. Let me walk you through how everything works.</p>
<h2>File Structure</h2>
<p>Here's the simplified structure of the project:</p>
<pre><code class="language-plaintext">├── app/
│   ├── page.js                # Homepage
│   ├── layout.js              # Root layout
│   └── post/[slug]/page.js    # Blog post pages

    ├── fonts/                 # Custom fonts
    └── images/                # Static images

<h2>Core Technologies</h2>
<p>The site is built using <code>Next.js 14</code> with React Server Components for better performance and SEO. Instead of using a CSS framework like Tailwind, I opted for pure CSS with variables for maximum control over styling and animations. All content is written in Markdown and processed using a combination of remark and rehype plugins for features like math equations and syntax highlighting.</p>
<p>For typography, I'm using a combination of Geist Mono (for code and general text) and Sentient (for headings) to create a clean, readable interface that works well for both articles and code.</p>
<h2>Blog System</h2>
<p>The blog system is built around markdown files with a robust processing pipeline. Each post is processed through remark and rehype, which handle various transformations. Here's how posts are defined:</p>
<pre><code class="language-javascript">// lib/api.js
const blogPosts = [
  {
    slug: "probability",
    title: "Probability",
    date: new Date("2024-09-19"),
    tags: ["machine learning", "math"],
    headerImage: "/header-images/probability.jpg",
    estimatedWordCount: 3000,
  },
];
</code></pre>

<p>The navigation is designed to be unobtrusive yet accessible. It includes a smart navbar that automatically hides when scrolling down:</p>
<pre><code class="language-javascript">const handleScroll = () => {
  const currentScrollY = window.scrollY;
  const scrollDelta = lastScrollY.current - currentScrollY;

  if (scrollDelta > 10 || currentScrollY &#x3C; 20) {

  }

</code></pre>

<h2>Special Features</h2>
<h3>Mathematics Support</h3>
<p>The site uses <code>KaTeX</code> for rendering math equations. The <code>LaTeX</code> content is processed during markdown conversion and styled to match the current theme.</p>
<h3>Code Highlighting</h3>
<p>Code blocks use <code>highlight.js</code> with theme-aware styling. Each block includes language detection and a copy button for better user experience.</p>
<h3>Dynamic Table of Contents</h3>
<p>The TOC component automatically generates a navigation menu from the post's headings:</p>
<pre><code class="language-javascript">const generateTOC = () => {
  const headers = document.querySelectorAll("h2, h3, h4, h5, h6");
  return Array.from(headers).map((header) => ({
    id: header.id,
    text: header.textContent,
    level: parseInt(header.tagName.charAt(1)),
  }));
};
</code></pre>
<h3>Performance Optimizations</h3>
<p>Several techniques keep the site fast and responsive:</p>
<ol>
<li>Server Components for improved initial load times</li>
<li>Dynamic imports for features not needed immediately</li>
<li>Font optimization with preloading and subsetting</li>
<li>Responsive images with lazy loading</li>
<li>Code splitting to reduce initial bundle size</li>
</ol>
