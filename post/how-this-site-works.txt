<h3>2. Real-time Analytics</h3>
<p>Every page incorporates an analytics system using Supabase's real-time capabilities:</p>
<pre><code class="language-javascript">export async function getStats(pageId) {
  const cached = getStatsFromCache(pageId);
  if (cached) return cached;

  try {
    const { data } = await supabase
      .from("page_stats")
      .select("views, likes")
      .eq("id", pageId)
      .single();

    const result = data || { views: 0, likes: 0 };
    statsCache.set(pageId, {
      data: result,
      timestamp: Date.now(),
    });
    return result;
  } catch {
    return { views: 0, likes: 0 };
  }
}
</code></pre>
<p>The system includes:</p>
<ul>
<li>Real-time view counting</li>
<li>Like functionality with optimistic updates</li>
<li>Efficient caching with timestamp-based invalidation</li>
<li>Multi-level storage (memory, localStorage, and database)</li>
<li>Debounced updates to prevent spam</li>
</ul>
