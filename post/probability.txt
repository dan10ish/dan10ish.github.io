b:T5411,<h2>1. Probability Fundamentals</h2>
<h3>1.1 Counting Principles</h3>
<h4>1.1.1 The Fundamental Counting Principle</h4>
<p>If one event can occur in <span class="math math-inline">m </span> ways, and another independent event can occur in <span class="math math-inline">n</span> ways, then the two events can occur together in <span class="math math-inline">m \times n</span> ways.</p>
<p>Example: If you have 3 shirts and 2 pairs of pants, you have <span class="math math-inline">3 \times 2 = 6</span> different outfit combinations.</p>
<h4>1.1.2 Permutations</h4>
<p>A permutation is an arrangement of objects where order matters.</p>
<p>Formula:</p>
<div class="math math-display">P(n,r) = \frac{n!}{(n-r)!}</div>
<p>Where <span class="math math-inline">n</span> is the total number of objects and <span class="math math-inline">r</span> is the number being arranged.</p>
<p>Example: How many ways can you arrange 3 books out of 5?</p>
<div class="math math-display">P(5,3) = \frac{5!}{(5-3)!} = \frac{5!}{2!} = \frac{120}{2} = 60</div>
<h4>1.1.3 Combinations</h4>
<p>A combination is a selection of objects where order doesn't matter.</p>
<p>Formula:</p>
<div class="math math-display">C(n,r) = \binom{n}{r} = \frac{n!}{r!(n-r)!}</div>
<p>Example: In how many ways can you select 3 books out of 5?</p>
<div class="math math-display">C(5,3) = \binom{5}{3} = \frac{5!}{3!(5-3)!} = \frac{5!}{3!2!} = \frac{120}{12} = 10</div>
<h3>1.2 Probability Axioms</h3>
<h4>1.2.1 Axiom 1: Non-negativity</h4>
<p>For any event <span class="math math-inline">A</span>, <span class="math math-inline">P(A) \geq 0</span></p>
<h4>1.2.2 Axiom 2: Probability of the entire sample space</h4>
<p><span class="math math-inline">P(S) = 1</span>, where <span class="math math-inline">S</span> is the sample space</p>
<h4>1.2.3 Axiom 3: Additivity</h4>
<p>For mutually exclusive events <span class="math math-inline">A</span> and <span class="math math-inline">B</span>, <span class="math math-inline">P(A \cup B) = P(A) + P(B)</span></p>
<h3>1.3 Basic Probability Concepts</h3>
<h4>1.3.1 Sample Space</h4>
<p>The set of all possible outcomes of an experiment.</p>
<p>Example: When rolling a die, <span class="math math-inline">S = \{1, 2, 3, 4, 5, 6\}</span></p>
<h4>1.3.2 Events</h4>
<p>An event is a subset of the sample space.</p>
<p>Example: The event of rolling an even number: <span class="math math-inline">E = \{2, 4, 6\}</span></p>
<h4>1.3.3 Probability of an Event</h4>
<div class="math math-display">P(A) = \frac{\text{Number of favorable outcomes}}{\text{Total number of possible outcomes}}</div>
<p>Example: <span class="math math-inline">P(\text{rolling an even number}) = \frac{3}{6} = \frac{1}{2}</span></p>
<h4>1.3.4 Conditional Probability</h4>
<p>The probability of an event <span class="math math-inline">A</span>, given that event <span class="math math-inline">B</span> has occurred.</p>
<p>Formula:</p>
<div class="math math-display">P(A|B) = \frac{P(A \cap B)}{P(B)}, \quad P(B) > 0</div>
<p>Example: In a deck of 52 cards, what's the probability of drawing a king, given that it's a face card?</p>
<div class="math math-display">P(\text{King | Face Card}) = \frac{P(\text{King} \cap \text{Face Card})}{P(\text{Face Card})} = \frac{4/52}{12/52} = \frac{1}{3}</div>
<h4>1.3.5 Independent Events</h4>
<p>Events <span class="math math-inline">A</span> and <span class="math math-inline">B</span> are independent if <span class="math math-inline">P(A \cap B) = P(A) \times P(B)</span></p>
<p>Example: The probability of getting heads on two coin flips:</p>
<div class="math math-display">P(H \cap H) = P(H) \times P(H) = \frac{1}{2} \times \frac{1}{2} = \frac{1}{4}</div>
<h4>1.3.6 Mutually Exclusive Events</h4>
<p>Events that cannot occur simultaneously. If <span class="math math-inline">A</span> and <span class="math math-inline">B</span> are mutually exclusive, <span class="math math-inline">P(A \cap B) = 0</span></p>
<p>Example: Rolling an even number and an odd number on a single die roll.</p>
<h3>1.4 Additional Important Concepts</h3>
<h4>1.4.1 Bayes' Theorem</h4>
<div class="math math-display">P(A|B) = \frac{P(B|A)P(A)}{P(B)} = \frac{P(B|A)P(A)}{P(B|A)P(A) + P(B|A^c)P(A^c)}</div>
<p>This theorem is crucial for updating probabilities based on new evidence.</p>
<h4>1.4.2 Law of Total Probability</h4>
<p>For mutually exclusive and exhaustive events <span class="math math-inline">B_1, B_2, ..., B_n</span>:</p>
<div class="math math-display">P(A) = \sum_{i=1}^n P(A|B_i)P(B_i)</div>
<p>This law is useful for calculating probabilities when multiple scenarios are possible.</p>
<h2>2. Random Variables and Probability Distributions</h2>
<h3>2.1 Random Variables</h3>
<p>A random variable is a variable whose possible values are outcomes of a random phenomenon.</p>
<h4>2.1.1 Discrete Random Variables</h4>
<p>Can only take on a countable number of distinct values.</p>
<p>Example: Number of heads in 3 coin flips (possible values: 0, 1, 2, 3)</p>
<h4>2.1.2 Continuous Random Variables</h4>
<p>Can take on any value within a given range.</p>
<p>Example: Height of a person (can be any real number within a realistic range)</p>
<h3>2.2 Probability Mass Function (PMF)</h3>
<p>For a discrete random variable <span class="math math-inline">X</span>, the PMF gives the probability that <span class="math math-inline">X</span> is exactly equal to some value <span class="math math-inline">x</span>.</p>
<p>Properties:</p>
<ol>
<li><span class="math math-inline">0 \leq P(X = x) \leq 1</span></li>
<li><span class="math math-inline">\sum_x P(X = x) = 1</span> (sum over all possible <span class="math math-inline">x</span>)</li>
</ol>
<h3>2.3 Probability Density Function (PDF)</h3>
<p>For a continuous random variable <span class="math math-inline">X</span>, the PDF <span class="math math-inline">f(x)</span> is used such that the probability of <span class="math math-inline">X</span> falling in an interval <span class="math math-inline">[a, b]</span> is given by the integral of <span class="math math-inline">f(x)</span> from <span class="math math-inline">a</span> to <span class="math math-inline">b</span>.</p>
<p>Properties:</p>
<ol>
<li><span class="math math-inline">f(x) \geq 0</span> for all <span class="math math-inline">x</span></li>
<li><span class="math math-inline">\int_{-\infty}^{\infty} f(x)dx = 1</span></li>
</ol>
<h3>2.4 Cumulative Distribution Function (CDF)</h3>
<p>The CDF <span class="math math-inline">F(x)</span> gives the probability that <span class="math math-inline">X</span> takes on a value less than or equal to <span class="math math-inline">x</span>.</p>
<p>For discrete <span class="math math-inline">X</span>: <span class="math math-inline">F(x) = P(X \leq x) = \sum_{t \leq x} P(X = t)</span>
For continuous <span class="math math-inline">X</span>: <span class="math math-inline">F(x) = P(X \leq x) = \int_{-\infty}^x f(t)dt</span></p>
<h3>2.5 Expected Value and Variance</h3>
<h4>2.5.1 Expected Value (Mean)</h4>
<p>For discrete <span class="math math-inline">X</span>: <span class="math math-inline">E(X) = \sum_x x P(X = x)</span>
For continuous <span class="math math-inline">X</span>: <span class="math math-inline">E(X) = \int_{-\infty}^{\infty} x f(x)dx</span></p>
<h4>2.5.2 Variance and Standard Deviation</h4>
<p>Variance: <span class="math math-inline">Var(X) = E[(X - E(X))^2] = E(X^2) - [E(X)]^2</span>
Standard Deviation: <span class="math math-inline">\sigma = \sqrt{Var(X)}</span></p>
<h3>2.6 Common Discrete Distributions</h3>
<h4>2.6.1 Bernoulli Distribution</h4>
<p>Models a single trial with two possible outcomes (success/failure).</p>
<ul>
<li>PMF: <span class="math math-inline">P(X = 1) = p</span>, <span class="math math-inline">P(X = 0) = 1-p</span></li>
<li>Mean: <span class="math math-inline">E(X) = p</span></li>
<li>Variance: <span class="math math-inline">Var(X) = p(1-p)</span></li>
</ul>
<h4>2.6.2 Binomial Distribution</h4>
<p>Models the number of successes in <span class="math math-inline">n</span> independent Bernoulli trials.</p>
<ul>
<li>PMF: <span class="math math-inline">P(X = k) = \binom{n}{k} p^k (1-p)^{n-k}</span></li>
<li>Mean: <span class="math math-inline">E(X) = np</span></li>
<li>Variance: <span class="math math-inline">Var(X) = np(1-p)</span></li>
</ul>
<h4>2.6.3 Poisson Distribution</h4>
<p>Models the number of events occurring in a fixed interval of time or space.</p>
<ul>
<li>PMF: <span class="math math-inline">P(X = k) = \frac{\lambda^k e^{-\lambda}}{k!}</span></li>
<li>Mean: <span class="math math-inline">E(X) = \lambda</span></li>
<li>Variance: <span class="math math-inline">Var(X) = \lambda</span></li>
</ul>
<h3>2.7 Common Continuous Distributions</h3>
<h4>2.7.1 Uniform Distribution</h4>
<p>All intervals of the same length on the distribution's support are equally probable.</p>
<ul>
<li>PDF: <span class="math math-inline">f(x) = \frac{1}{b-a}</span> for <span class="math math-inline">a \leq x \leq b</span></li>
<li>Mean: <span class="math math-inline">E(X) = \frac{a+b}{2}</span></li>
<li>Variance: <span class="math math-inline">Var(X) = \frac{(b-a)^2}{12}</span></li>
</ul>
<h4>2.7.2 Normal (Gaussian) Distribution</h4>
<p>Bell-shaped distribution defined by its mean <span class="math math-inline">\mu</span> and standard deviation <span class="math math-inline">\sigma</span>.</p>
<ul>
<li>PDF:
<div class="math math-display">f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}</div>
</li>
<li>Mean: <span class="math math-inline">E(X) = \mu</span></li>
<li>Variance: <span class="math math-inline">Var(X) = \sigma^2</span></li>
</ul>
<h4>2.7.3 Exponential Distribution</h4>
<p>Models the time between events in a Poisson process.</p>
<ul>
<li>PDF: <span class="math math-inline">f(x) = \lambda e^{-\lambda x}</span> for <span class="math math-inline">x \geq 0</span></li>
<li>Mean: <span class="math math-inline">E(X) = \frac{1}{\lambda}</span></li>
<li>Variance: <span class="math math-inline">Var(X) = \frac{1}{\lambda^2}</span></li>
</ul>
<h2>3. Joint Probability Distributions, Covariance, and Correlation</h2>
<h3>3.1 Joint Probability Distributions</h3>
<h4>3.1.1 Joint Probability Mass Function (Joint PMF)</h4>
<p>For discrete random variables <span class="math math-inline">X</span> and <span class="math math-inline">Y</span>:</p>
<div class="math math-display">P(X = x, Y = y) = P(X = x \text{ and } Y = y)</div>
<p>Properties:</p>
<ol>
<li><span class="math math-inline">P(x, y) \geq 0</span> for all <span class="math math-inline">x</span> and <span class="math math-inline">y</span></li>
<li><span class="math math-inline">\sum_x \sum_y P(x, y) = 1</span> (sum over all possible <span class="math math-inline">x</span> and <span class="math math-inline">y</span>)</li>
</ol>
<h4>3.1.2 Joint Probability Density Function (Joint PDF)</h4>
<p>For continuous random variables <span class="math math-inline">X</span> and <span class="math math-inline">Y</span>, the joint PDF <span class="math math-inline">f(x, y)</span> is used such that:</p>
<div class="math math-display">P(a \leq X \leq b, c \leq Y \leq d) = \int_a^b \int_c^d f(x, y) \, dy \, dx</div>
<p>Properties:</p>
<ol>
<li><span class="math math-inline">f(x, y) \geq 0</span> for all <span class="math math-inline">x</span> and <span class="math math-inline">y</span></li>
<li><span class="math math-inline">\int_{-\infty}^{\infty} \int_{-\infty}^{\infty} f(x, y) \, dx \, dy = 1</span></li>
</ol>
<h3>3.2 Marginal Distributions</h3>
<h4>3.2.1 Marginal PMF</h4>
<p>For discrete variables:</p>
<div class="math math-display">P(X = x) = \sum_y P(X = x, Y = y)</div>
<h4>3.2.2 Marginal PDF</h4>
<p>For continuous variables:</p>
<div class="math math-display">f_X(x) = \int_{-\infty}^{\infty} f(x, y) \, dy</div>
<h3>3.3 Conditional Distributions</h3>
<h4>3.3.1 Conditional PMF</h4>
<p>For discrete variables:</p>
<div class="math math-display">P(Y = y | X = x) = \frac{P(X = x, Y = y)}{P(X = x)}</div>
<h4>3.3.2 Conditional PDF</h4>
<p>For continuous variables:</p>
<div class="math math-display">f_{Y|X}(y|x) = \frac{f(x, y)}{f_X(x)}</div>
<h3>3.4 Independence of Random Variables</h3>
<p>Random variables <span class="math math-inline">X</span> and <span class="math math-inline">Y</span> are independent if:</p>
<div class="math math-display">P(X = x, Y = y) = P(X = x) \times P(Y = y)</div>
<p>for all <span class="math math-inline">x</span> and <span class="math math-inline">y</span></p>
<p>For continuous variables:</p>
<div class="math math-display">f(x, y) = f_X(x) \times f_Y(y)</div>
<p>for all <span class="math math-inline">x</span> and <span class="math math-inline">y</span></p>
<h3>3.5 Covariance</h3>
<p>Covariance measures the joint variability of two random variables.</p>
<div class="math math-display">Cov(X, Y) = E[(X - E[X])(Y - E[Y])] = E[XY] - E[X]E[Y]</div>
<p>Properties:</p>
<ol>
<li><span class="math math-inline">Cov(X, X) = Var(X)</span></li>
<li><span class="math math-inline">Cov(X, Y) = Cov(Y, X)</span></li>
<li><span class="math math-inline">Cov(aX, bY) = ab \times Cov(X, Y)</span></li>
</ol>
<h3>3.6 Correlation</h3>
<p>Correlation is a normalized measure of the linear relationship between two variables.</p>
<p>Correlation coefficient:</p>
<div class="math math-display">\rho = \frac{Cov(X, Y)}{\sigma_X \sigma_Y}</div>
<p>Properties:</p>
<ol>
<li><span class="math math-inline">-1 \leq \rho \leq 1</span></li>
<li><span class="math math-inline">\rho = 1</span> or <span class="math math-inline">-1</span> indicates perfect linear relationship</li>
<li><span class="math math-inline">\rho = 0</span> indicates no linear relationship (but doesn't rule out other types of relationships)</li>
</ol>
<h2>4. Limit Theorems and Sampling Distributions</h2>
<h3>4.1 Law of Large Numbers (LLN)</h3>
<h4>4.1.1 Weak Law of Large Numbers</h4>
<p>For a sequence of i.i.d. random variables <span class="math math-inline">X_1, X_2, ..., X_n</span> with <span class="math math-inline">E[X_i] = \mu</span>,</p>
<div class="math math-display">P(|\bar{X}_n - \mu| > \varepsilon) \to 0 \text{ as } n \to \infty, \text{ for any } \varepsilon > 0</div>
<p>Where <span class="math math-inline">\bar{X}_n</span> is the sample mean of <span class="math math-inline">n</span> observations.</p>
<h4>4.1.2 Strong Law of Large Numbers</h4>
<div class="math math-display">P(\lim_{n \to \infty} \bar{X}_n = \mu) = 1</div>
<p>The strong law implies that the sample mean almost surely converges to the expected value.</p>
<h3>4.2 Central Limit Theorem (CLT)</h3>
<p>If <span class="math math-inline">X_1, X_2, ..., X_n</span> are i.i.d. random variables with <span class="math math-inline">E[X_i] = \mu</span> and <span class="math math-inline">Var(X_i) = \sigma^2</span>, then:</p>
<div class="math math-display">\frac{\bar{X}_n - \mu}{\sigma/\sqrt{n}} \xrightarrow{d} N(0,1) \text{ as } n \to \infty</div>
<p>Where <span class="math math-inline">\bar{X}_n</span> is the sample mean, and <span class="math math-inline">N(0,1)</span> is the standard normal distribution.</p>
<h3>4.3 Sampling Distributions</h3>
<h4>4.3.1 Sampling Distribution of the Mean</h4>
<p>For a population with mean <span class="math math-inline">\mu</span> and standard deviation <span class="math math-inline">\sigma</span>:</p>
<ol>
<li><span class="math math-inline">E(\bar{X}) = \mu</span></li>
<li><span class="math math-inline">\sigma_{\bar{X}} = \frac{\sigma}{\sqrt{n}}</span>, where <span class="math math-inline">n</span> is the sample size</li>
<li>If the population is normally distributed, <span class="math math-inline">\bar{X} \sim N(\mu, \frac{\sigma^2}{n})</span> for any sample size</li>
<li>If the population is not normal but <span class="math math-inline">n</span> is large (typically <span class="math math-inline">n \geq 30</span>), <span class="math math-inline">\bar{X}</span> is approximately normal by CLT</li>
</ol>
<h4>4.3.2 Sampling Distribution of the Proportion</h4>
<p>For a population proportion <span class="math math-inline">p</span>:</p>
<ol>
<li><span class="math math-inline">E(\hat{p}) = p</span></li>
<li><span class="math math-inline">\sigma_{\hat{p}} = \sqrt{\frac{p(1-p)}{n}}</span>, where <span class="math math-inline">n</span> is the sample size</li>
<li>For large <span class="math math-inline">n</span>, <span class="math math-inline">\hat{p}</span> is approximately normal with <span class="math math-inline">\mu = p</span> and <span class="math math-inline">\sigma = \sqrt{\frac{p(1-p)}{n}}</span></li>
</ol>
<h3>4.4 Student's t-Distribution</h3>
<p>When the population standard deviation is unknown and estimated from the sample, we use the t-distribution instead of the normal distribution.</p>
<p>The t-distribution has heavier tails than the normal distribution and approaches the normal distribution as degrees of freedom increase.</p>
<p>Degrees of freedom = <span class="math math-inline">n - 1</span>, where <span class="math math-inline">n</span> is the sample size.</p>
<h3>4.5 Chi-Square Distribution</h3>
<p>The chi-square distribution is used in hypothesis testing and constructing confidence intervals for population variance.</p>
<p>If <span class="math math-inline">Z_1, Z_2, ..., Z_k</span> are independent standard normal random variables, then the sum of their squares follows a chi-square distribution with <span class="math math-inline">k</span> degrees of freedom:</p>

<h2>5. Hypothesis Testing and Confidence Intervals</h2>
<h3>5.1 Hypothesis Testing</h3>
<p>Hypothesis testing is a method of statistical inference used to decide whether there is enough evidence in a sample of data to infer that a certain condition is true for the entire population.</p>
<h4>5.1.1 Steps in Hypothesis Testing</h4>
<ol>
<li>State the null (<span class="math math-inline">H_0</span>) and alternative (<span class="math math-inline">H_1</span>) hypotheses</li>
<li>Choose the significance level (<span class="math math-inline">\alpha</span>)</li>
<li>Calculate the test statistic</li>
<li>Determine the critical value or p-value</li>
<li>Make a decision to reject or fail to reject <span class="math math-inline">H_0</span></li>
</ol>
<h4>5.1.2 Types of Errors</h4>
<ul>
<li>Type I Error: Rejecting <span class="math math-inline">H_0</span> when it's true (probability = <span class="math math-inline">\alpha</span>)</li>
<li>Type II Error: Failing to reject <span class="math math-inline">H_0</span> when it's false (probability = <span class="math math-inline">\beta</span>)</li>
<li>Power of the test = <span class="math math-inline">1 - \beta</span></li>
</ul>
<h3>5.2 Z-Test</h3>
<p>Used when the population standard deviation is known and the sample size is large (<span class="math math-inline">n \geq 30</span>).</p>
<h4>5.2.1 One-Sample Z-Test</h4>
<p>Test statistic:</p>
<div class="math math-display">Z = \frac{\bar{X} - \mu_0}{\sigma / \sqrt{n}}</div>
<p>Where <span class="math math-inline">\mu_0</span> is the hypothesized population mean</p>
<h4>5.2.2 Two-Sample Z-Test</h4>
<p>Test statistic:</p>
<div class="math math-display">Z = \frac{(\bar{X}_1 - \bar{X}_2) - (\mu_1 - \mu_2)}{\sqrt{\frac{\sigma_1^2}{n_1} + \frac{\sigma_2^2}{n_2}}}</div>
<h3>5.3 T-Test</h3>
<p>Used when the population standard deviation is unknown and/or the sample size is small (<span class="math math-inline">n &#x3C; 30</span>).</p>
<h4>5.3.1 One-Sample T-Test</h4>
<p>Test statistic:</p>
<div class="math math-display">t = \frac{\bar{X} - \mu_0}{s / \sqrt{n}}</div>
<p>Where <span class="math math-inline">s</span> is the sample standard deviation</p>
<h4>5.3.2 Two-Sample T-Test</h4>
<p>For equal variances:</p>
<div class="math math-display">t = \frac{(\bar{X}_1 - \bar{X}_2) - (\mu_1 - \mu_2)}{s_p \sqrt{\frac{1}{n_1} + \frac{1}{n_2}}}</div>
<p>Where <span class="math math-inline">s_p</span> is the pooled standard deviation</p>
<p>For unequal variances (Welch's t-test):</p>
<div class="math math-display">t = \frac{(\bar{X}_1 - \bar{X}_2) - (\mu_1 - \mu_2)}{\sqrt{\frac{s_1^2}{n_1} + \frac{s_2^2}{n_2}}}</div>
<h3>5.4 Chi-Square Test</h3>
<p>Used for categorical data to test goodness of fit or independence.</p>
<h4>5.4.1 Goodness of Fit Test</h4>
<p>Test statistic:</p>
<div class="math math-display">\chi^2 = \sum_{i=1}^k \frac{(O_i - E_i)^2}{E_i}</div>
<p>Where <span class="math math-inline">O_i</span> is the observed frequency and <span class="math math-inline">E_i</span> is the expected frequency</p>
<h4>5.4.2 Test of Independence</h4>
<p>Test statistic:</p>
<div class="math math-display">\chi^2 = \sum_{i=1}^r \sum_{j=1}^c \frac{(O_{ij} - E_{ij})^2}{E_{ij}}</div>
<p>Where <span class="math math-inline">O_{ij}</span> and <span class="math math-inline">E_{ij}</span> are the observed and expected frequencies in cell <span class="math math-inline">(i,j)</span></p>
<h3>5.5 Confidence Intervals</h3>
<p>A confidence interval provides a range of values that is likely to contain the population parameter with a certain level of confidence.</p>
<h4>5.5.1 Confidence Interval for Mean (known <span class="math math-inline">\sigma</span>)</h4>
<div class="math math-display">\text{CI}: \bar{X} \pm z_{\alpha/2} \frac{\sigma}{\sqrt{n}}</div>
<h4>5.5.2 Confidence Interval for Mean (unknown <span class="math math-inline">\sigma</span>)</h4>
<div class="math math-display">\text{CI}: \bar{X} \pm t_{\alpha/2} \frac{s}{\sqrt{n}}</div>
<h4>5.5.3 Confidence Interval for Proportion</h4>
<div class="math math-display">\text{CI}: \hat{p} \pm z_{\alpha/2} \sqrt{\frac{\hat{p}(1-\hat{p})}{n}}</div>

<h3>1.1 Counting Principles</h3>
<h4>1.1.1 The Fundamental Counting Principle</h4>
<p>If one event can occur in <span class="math math-inline">m </span> ways, and another independent event can occur in <span class="math math-inline">n</span> ways, then the two events can occur together in <span class="math math-inline">m \times n</span> ways.</p>
<p>Example: If you have 3 shirts and 2 pairs of pants, you have <span class="math math-inline">3 \times 2 = 6</span> different outfit combinations.</p>
<h4>1.1.2 Permutations</h4>
<p>A permutation is an arrangement of objects where order matters.</p>
<p>Formula:</p>
<div class="math math-display">P(n,r) = \frac{n!}{(n-r)!}</div>
<p>Where <span class="math math-inline">n</span> is the total number of objects and <span class="math math-inline">r</span> is the number being arranged.</p>
<p>Example: How many ways can you arrange 3 books out of 5?</p>
<div class="math math-display">P(5,3) = \frac{5!}{(5-3)!} = \frac{5!}{2!} = \frac{120}{2} = 60</div>
<h4>1.1.3 Combinations</h4>
<p>A combination is a selection of objects where order doesn't matter.</p>
<p>Formula:</p>
<div class="math math-display">C(n,r) = \binom{n}{r} = \frac{n!}{r!(n-r)!}</div>
<p>Example: In how many ways can you select 3 books out of 5?</p>
<div class="math math-display">C(5,3) = \binom{5}{3} = \frac{5!}{3!(5-3)!} = \frac{5!}{3!2!} = \frac{120}{12} = 10</div>
<h3>1.2 Probability Axioms</h3>
<h4>1.2.1 Axiom 1: Non-negativity</h4>
<p>For any event <span class="math math-inline">A</span>, <span class="math math-inline">P(A) \geq 0</span></p>
<h4>1.2.2 Axiom 2: Probability of the entire sample space</h4>
<p><span class="math math-inline">P(S) = 1</span>, where <span class="math math-inline">S</span> is the sample space</p>
<h4>1.2.3 Axiom 3: Additivity</h4>
<p>For mutually exclusive events <span class="math math-inline">A</span> and <span class="math math-inline">B</span>, <span class="math math-inline">P(A \cup B) = P(A) + P(B)</span></p>
<h3>1.3 Basic Probability Concepts</h3>
<h4>1.3.1 Sample Space</h4>
<p>The set of all possible outcomes of an experiment.</p>
<p>Example: When rolling a die, <span class="math math-inline">S = \{1, 2, 3, 4, 5, 6\}</span></p>
<h4>1.3.2 Events</h4>
<p>An event is a subset of the sample space.</p>
<p>Example: The event of rolling an even number: <span class="math math-inline">E = \{2, 4, 6\}</span></p>
<h4>1.3.3 Probability of an Event</h4>
<div class="math math-display">P(A) = \frac{\text{Number of favorable outcomes}}{\text{Total number of possible outcomes}}</div>
<p>Example: <span class="math math-inline">P(\text{rolling an even number}) = \frac{3}{6} = \frac{1}{2}</span></p>
<h4>1.3.4 Conditional Probability</h4>
<p>The probability of an event <span class="math math-inline">A</span>, given that event <span class="math math-inline">B</span> has occurred.</p>
<p>Formula:</p>
<div class="math math-display">P(A|B) = \frac{P(A \cap B)}{P(B)}, \quad P(B) > 0</div>
<p>Example: In a deck of 52 cards, what's the probability of drawing a king, given that it's a face card?</p>
<div class="math math-display">P(\text{King | Face Card}) = \frac{P(\text{King} \cap \text{Face Card})}{P(\text{Face Card})} = \frac{4/52}{12/52} = \frac{1}{3}</div>
<h4>1.3.5 Independent Events</h4>
<p>Events <span class="math math-inline">A</span> and <span class="math math-inline">B</span> are independent if <span class="math math-inline">P(A \cap B) = P(A) \times P(B)</span></p>
<p>Example: The probability of getting heads on two coin flips:</p>
<div class="math math-display">P(H \cap H) = P(H) \times P(H) = \frac{1}{2} \times \frac{1}{2} = \frac{1}{4}</div>
<h4>1.3.6 Mutually Exclusive Events</h4>
<p>Events that cannot occur simultaneously. If <span class="math math-inline">A</span> and <span class="math math-inline">B</span> are mutually exclusive, <span class="math math-inline">P(A \cap B) = 0</span></p>
<p>Example: Rolling an even number and an odd number on a single die roll.</p>
<h3>1.4 Additional Important Concepts</h3>
<h4>1.4.1 Bayes' Theorem</h4>
<div class="math math-display">P(A|B) = \frac{P(B|A)P(A)}{P(B)} = \frac{P(B|A)P(A)}{P(B|A)P(A) + P(B|A^c)P(A^c)}</div>
<p>This theorem is crucial for updating probabilities based on new evidence.</p>
<h4>1.4.2 Law of Total Probability</h4>
<p>For mutually exclusive and exhaustive events <span class="math math-inline">B_1, B_2, ..., B_n</span>:</p>
<div class="math math-display">P(A) = \sum_{i=1}^n P(A|B_i)P(B_i)</div>
<p>This law is useful for calculating probabilities when multiple scenarios are possible.</p>
<h2>2. Random Variables and Probability Distributions</h2>
<h3>2.1 Random Variables</h3>
<p>A random variable is a variable whose possible values are outcomes of a random phenomenon.</p>
<h4>2.1.1 Discrete Random Variables</h4>
<p>Can only take on a countable number of distinct values.</p>
<p>Example: Number of heads in 3 coin flips (possible values: 0, 1, 2, 3)</p>
<h4>2.1.2 Continuous Random Variables</h4>
<p>Can take on any value within a given range.</p>
<p>Example: Height of a person (can be any real number within a realistic range)</p>
<h3>2.2 Probability Mass Function (PMF)</h3>
<p>For a discrete random variable <span class="math math-inline">X</span>, the PMF gives the probability that <span class="math math-inline">X</span> is exactly equal to some value <span class="math math-inline">x</span>.</p>
<p>Properties:</p>
<ol>
<li><span class="math math-inline">0 \leq P(X = x) \leq 1</span></li>
<li><span class="math math-inline">\sum_x P(X = x) = 1</span> (sum over all possible <span class="math math-inline">x</span>)</li>
</ol>
<h3>2.3 Probability Density Function (PDF)</h3>
<p>For a continuous random variable <span class="math math-inline">X</span>, the PDF <span class="math math-inline">f(x)</span> is used such that the probability of <span class="math math-inline">X</span> falling in an interval <span class="math math-inline">[a, b]</span> is given by the integral of <span class="math math-inline">f(x)</span> from <span class="math math-inline">a</span> to <span class="math math-inline">b</span>.</p>
<p>Properties:</p>
<ol>
<li><span class="math math-inline">f(x) \geq 0</span> for all <span class="math math-inline">x</span></li>
<li><span class="math math-inline">\int_{-\infty}^{\infty} f(x)dx = 1</span></li>
</ol>
<h3>2.4 Cumulative Distribution Function (CDF)</h3>
<p>The CDF <span class="math math-inline">F(x)</span> gives the probability that <span class="math math-inline">X</span> takes on a value less than or equal to <span class="math math-inline">x</span>.</p>
<p>For discrete <span class="math math-inline">X</span>: <span class="math math-inline">F(x) = P(X \leq x) = \sum_{t \leq x} P(X = t)</span>
For continuous <span class="math math-inline">X</span>: <span class="math math-inline">F(x) = P(X \leq x) = \int_{-\infty}^x f(t)dt</span></p>
<h3>2.5 Expected Value and Variance</h3>
<h4>2.5.1 Expected Value (Mean)</h4>
<p>For discrete <span class="math math-inline">X</span>: <span class="math math-inline">E(X) = \sum_x x P(X = x)</span>
For continuous <span class="math math-inline">X</span>: <span class="math math-inline">E(X) = \int_{-\infty}^{\infty} x f(x)dx</span></p>
<h4>2.5.2 Variance and Standard Deviation</h4>

<h2>4. Limit Theorems and Sampling Distributions</h2>
<h3>4.1 Law of Large Numbers (LLN)</h3>
<h4>4.1.1 Weak Law of Large Numbers</h4>
<p>For a sequence of i.i.d. random variables <span class="math math-inline">X_1, X_2, ..., X_n</span> with <span class="math math-inline">E[X_i] = \mu</span>,</p>
<div class="math math-display">P(|\bar{X}_n - \mu| > \varepsilon) \to 0 \text{ as } n \to \infty, \text{ for any } \varepsilon > 0</div>
<p>Where <span class="math math-inline">\bar{X}_n</span> is the sample mean of <span class="math math-inline">n</span> observations.</p>
<h4>4.1.2 Strong Law of Large Numbers</h4>
<div class="math math-display">P(\lim_{n \to \infty} \bar{X}_n = \mu) = 1</div>
<p>The strong law implies that the sample mean almost surely converges to the expected value.</p>
<h3>4.2 Central Limit Theorem (CLT)</h3>
<p>If <span class="math math-inline">X_1, X_2, ..., X_n</span> are i.i.d. random variables with <span class="math math-inline">E[X_i] = \mu</span> and <span class="math math-inline">Var(X_i) = \sigma^2</span>, then:</p>
<div class="math math-display">\frac{\bar{X}_n - \mu}{\sigma/\sqrt{n}} \xrightarrow{d} N(0,1) \text{ as } n \to \infty</div>
<p>Where <span class="math math-inline">\bar{X}_n</span> is the sample mean, and <span class="math math-inline">N(0,1)</span> is the standard normal distribution.</p>
<h3>4.3 Sampling Distributions</h3>
<h4>4.3.1 Sampling Distribution of the Mean</h4>
<p>For a population with mean <span class="math math-inline">\mu</span> and standard deviation <span class="math math-inline">\sigma</span>:</p>
<ol>
<li><span class="math math-inline">E(\bar{X}) = \mu</span></li>
<li><span class="math math-inline">\sigma_{\bar{X}} = \frac{\sigma}{\sqrt{n}}</span>, where <span class="math math-inline">n</span> is the sample size</li>
<li>If the population is normally distributed, <span class="math math-inline">\bar{X} \sim N(\mu, \frac{\sigma^2}{n})</span> for any sample size</li>
<li>If the population is not normal but <span class="math math-inline">n</span> is large (typically <span class="math math-inline">n \geq 30</span>), <span class="math math-inline">\bar{X}</span> is approximately normal by CLT</li>
</ol>
<h4>4.3.2 Sampling Distribution of the Proportion</h4>
<p>For a population proportion <span class="math math-inline">p</span>:</p>
<ol>
<li><span class="math math-inline">E(\hat{p}) = p</span></li>
<li><span class="math math-inline">\sigma_{\hat{p}} = \sqrt{\frac{p(1-p)}{n}}</span>, where <span class="math math-inline">n</span> is the sample size</li>
<li>For large <span class="math math-inline">n</span>, <span class="math math-inline">\hat{p}</span> is approximately normal with <span class="math math-inline">\mu = p</span> and <span class="math math-inline">\sigma = \sqrt{\frac{p(1-p)}{n}}</span></li>
</ol>
<h3>4.4 Student's t-Distribution</h3>
<p>When the population standard deviation is unknown and estimated from the sample, we use the t-distribution instead of the normal distribution.</p>
<p>The t-distribution has heavier tails than the normal distribution and approaches the normal distribution as degrees of freedom increase.</p>
<p>Degrees of freedom = <span class="math math-inline">n - 1</span>, where <span class="math math-inline">n</span> is the sample size.</p>
<h3>4.5 Chi-Square Distribution</h3>
<p>The chi-square distribution is used in hypothesis testing and constructing confidence intervals for population variance.</p>
<p>If <span class="math math-inline">Z_1, Z_2, ..., Z_k</span> are independent standard normal random variables, then the sum of their squares follows a chi-square distribution with <span class="math math-inline">k</span> degrees of freedom:</p>
<div class="math math-display">\sum_{i=1}^k Z_i^2 \sim \chi^2_k</div>
<h2>5. Hypothesis Testing and Confidence Intervals</h2>
<h3>5.1 Hypothesis Testing</h3>
<p>Hypothesis testing is a method of statistical inference used to decide whether there is enough evidence in a sample of data to infer that a certain condition is true for the entire population.</p>
<h4>5.1.1 Steps in Hypothesis Testing</h4>
<ol>
<li>State the null (<span class="math math-inline">H_0</span>) and alternative (<span class="math math-inline">H_1</span>) hypotheses</li>
<li>Choose the significance level (<span class="math math-inline">\alpha</span>)</li>
<li>Calculate the test statistic</li>
<li>Determine the critical value or p-value</li>
<li>Make a decision to reject or fail to reject <span class="math math-inline">H_0</span></li>
</ol>
<h4>5.1.2 Types of Errors</h4>
<ul>
<li>Type I Error: Rejecting <span class="math math-inline">H_0</span> when it's true (probability = <span class="math math-inline">\alpha</span>)</li>
<li>Type II Error: Failing to reject <span class="math math-inline">H_0</span> when it's false (probability = <span class="math math-inline">\beta</span>)</li>
<li>Power of the test = <span class="math math-inline">1 - \beta</span></li>
</ul>
<h3>5.2 Z-Test</h3>
<p>Used when the population standard deviation is known and the sample size is large (<span class="math math-inline">n \geq 30</span>).</p>
<h4>5.2.1 One-Sample Z-Test</h4>
<p>Test statistic:</p>
<div class="math math-display">Z = \frac{\bar{X} - \mu_0}{\sigma / \sqrt{n}}</div>
<p>Where <span class="math math-inline">\mu_0</span> is the hypothesized population mean</p>
<h4>5.2.2 Two-Sample Z-Test</h4>
<p>Test statistic:</p>
<div class="math math-display">Z = \frac{(\bar{X}_1 - \bar{X}_2) - (\mu_1 - \mu_2)}{\sqrt{\frac{\sigma_1^2}{n_1} + \frac{\sigma_2^2}{n_2}}}</div>
<h3>5.3 T-Test</h3>
<p>Used when the population standard deviation is unknown and/or the sample size is small (<span class="math math-inline">n &#x3C; 30</span>).</p>
<h4>5.3.1 One-Sample T-Test</h4>
<p>Test statistic:</p>
<div class="math math-display">t = \frac{\bar{X} - \mu_0}{s / \sqrt{n}}</div>
<p>Where <span class="math math-inline">s</span> is the sample standard deviation</p>
<h4>5.3.2 Two-Sample T-Test</h4>
<p>For equal variances:</p>
<div class="math math-display">t = \frac{(\bar{X}_1 - \bar{X}_2) - (\mu_1 - \mu_2)}{s_p \sqrt{\frac{1}{n_1} + \frac{1}{n_2}}}</div>
<p>Where <span class="math math-inline">s_p</span> is the pooled standard deviation</p>
<p>For unequal variances (Welch's t-test):</p>
<div class="math math-display">t = \frac{(\bar{X}_1 - \bar{X}_2) - (\mu_1 - \mu_2)}{\sqrt{\frac{s_1^2}{n_1} + \frac{s_2^2}{n_2}}}</div>
<h3>5.4 Chi-Square Test</h3>
<p>Used for categorical data to test goodness of fit or independence.</p>
<h4>5.4.1 Goodness of Fit Test</h4>
<p>Test statistic:</p>
<div class="math math-display">\chi^2 = \sum_{i=1}^k \frac{(O_i - E_i)^2}{E_i}</div>
<p>Where <span class="math math-inline">O_i</span> is the observed frequency and <span class="math math-inline">E_i</span> is the expected frequency</p>
<h4>5.4.2 Test of Independence</h4>
<p>Test statistic:</p>
<div class="math math-display">\chi^2 = \sum_{i=1}^r \sum_{j=1}^c \frac{(O_{ij} - E_{ij})^2}{E_{ij}}</div>
<p>Where <span class="math math-inline">O_{ij}</span> and <span class="math math-inline">E_{ij}</span> are the observed and expected frequencies in cell <span class="math math-inline">(i,j)</span></p>
<h3>5.5 Confidence Intervals</h3>
<p>A confidence interval provides a range of values that is likely to contain the population parameter with a certain level of confidence.</p>
<h4>5.5.1 Confidence Interval for Mean (known <span class="math math-inline">\sigma</span>)</h4>
<div class="math math-display">\text{CI}: \bar{X} \pm z_{\alpha/2} \frac{\sigma}{\sqrt{n}}</div>
<h4>5.5.2 Confidence Interval for Mean (unknown <span class="math math-inline">\sigma</span>)</h4>
<div class="math math-display">\text{CI}: \bar{X} \pm t_{\alpha/2} \frac{s}{\sqrt{n}}</div>
<h4>5.5.3 Confidence Interval for Proportion</h4>
<div class="math math-display">\text{CI}: \hat{p} \pm z_{\alpha/2} \sqrt{\frac{\hat{p}(1-\hat{p})}{n}}</div>
